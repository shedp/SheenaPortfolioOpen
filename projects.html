Projects

Upper Hand Poker

A poker game with a twist! - you can see a player's hand! It is built with Vite, tailwind CSS with a Flask server and a postgreSQL database. We use Flask-Sockets and Redux in building the game flow. Assets were a mixture of DALL-E and Photoshop.

URL Shortener

A URL shortener for lovers of pink! This is a URL shorten created with Python (Flask) and Jinga for the frontend. It uses an SQL database (SQLAlchemy).

Ye Olde Quiz

Ye Old Quiz brings the pub quiz to your homes. This retro-style multiplayer quiz game inspired by a classic pub quiz. It uses Socket.io for the live multi-player functionality, and a React.js frontend with a Node.js for the backend.

Git Tracker

Octodog > Octocat. With Git Tracker, you can search for Github users and view their stats such as repos and languages. It utilises the Github API and React for the frontend. It uses CharJS to visualize the language usage by the user. (Dogs are better trackers!)

Habitual

Turn your rituals into good habits. Habitual is a habit tracker created using NodeJS, ExpressJS, authentication (bycrypt), and a postgreSQL database. The frontend is made using vanilla CSS and Javascript.

Liberty

Free your unconventional ideas with Liberty - an anonymous blogging app created with a mongoDB and Mongoose. It was created with vanilla CSS and Javascript.

preVent

Vent to Prevent! This is an anonymous blogging app that encourages you to vent out your frustrations. It utilises a GIF api and was created with vanilla CSS and Javascript.
